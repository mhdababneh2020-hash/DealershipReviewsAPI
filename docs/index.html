🚗 Dealership Reviews API

A RESTful API built with ASP.NET Core, Entity Framework & JWT Authentication


View Live API Docs →

🔐 Authentication

POST /api/Auth/register Register new user
POST /api/Auth/login Login & get JWT token

🏢 Dealerships

GET /api/Dealerships Get all (filter by city/state)
GET /api/Dealerships/{id} Get dealership by ID
POST /api/Dealerships Create new dealership
PUT /api/Dealerships/{id} Update dealership
DELETE /api/Dealerships/{id} Delete dealership

⭐ Reviews

GET /api/Reviews Get all reviews
GET /api/Reviews/dealer/{id} Get reviews by dealership
POST /api/Reviews Submit a review
DELETE /api/Reviews/{id} Delete review

🛠️ Tech Stack

C# ASP.NET Core 9 Entity Framework Core SQLite JWT Authentication Swagger UI Docker Render
